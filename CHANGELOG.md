## [1.0.0] - 2025-01-21

### Added
- Initial release of Novobhumi website
- React frontend with TypeScript and Tailwind CSS
- Express.js backend with Prisma ORM
- Admin panel for content management
- Real-time updates via Server-Sent Events
- Responsive design for all devices

### Features
- **Hero Section**: Animated landing with product showcase
- **Benefits Section**: 12 key benefits with professional design
- **Products Section**: Product catalog with buy buttons
- **Testimonials**: Customer reviews carousel
- **Comparison Table**: Traditional vs Novobhumi cocopeat
- **Admin Panel**: Secret /mayur-admin route for management

### Security
- Bcrypt password hashing
- Session-based authentication
- CORS protection
- Input validation
