## Prompts for Google Nano / Gemini image generation

Copy the prompt exactly, generate one image per filename, then drop the PNG here.

## Tips for best results

- Use a **1:1 or 4:3 aspect ratio** so the card grid stays tidy.
- Keep the **background white or transparent** so the cards blend in.
- Avoid tiny text; the cards scale down on mobile.
- After dropping the files, the site shows the image automatically — no code changes needed.
